# Site Pro en PHP

## Prérequis
- Laragon (Apache + MySQL + PHP)
- Visual Studio Code

## Installation
1. Cloner le projet dans `C:\laragon\www\site-pro`
2. Démarrer Laragon
3. Importer la base de données `site.sql` via phpMyAdmin
4. Accéder au site : [http://site-pro.test](http://site-pro.test)
